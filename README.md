# 로또

## 기능 구현 사항

☑️ 입력 받기
  - 로또 구입 금액
    - 예외: 1000원으로 나눠떨어지지 않는 금액 → IllegalArgumentException
  - 당첨 번호
    - 예외 → IllegalArgumentException
      - `,`가 아닌 구분자가 존재하는 경우
      - 1~45 사이의 번호가 아닌 경우
      - 숫자가 6개가 아닌 경우
  - 보너스 번호
    - 예외 → IllegalArgumentException
      - 1~45 사이의 번호가 아닌 경우

☑️ 로또 번호 추출

☑️ 로또 번호와 당첨 번호 비교 후, 당첨 여부 정리 

☑️ 출력 
  - 발행한 로또 수량
  - 발행한 로또 번호
  - 당첨 내역
  - 수익률 (= 당첨금액 / 구입금액 * 100)
